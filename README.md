# Qiny Api
Api of Qiny invoicing app.
The client facing app is located at [Qiny Client](https://github.com/selasiehanson/qiny_client) 


## Features
* Clients
* Taxes
* Invoices
* Products
* Signup and Authentication
* Account Chooser & Multi Tenancy
